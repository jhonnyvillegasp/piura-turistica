# Sitio Web Turístico de Piura

Sitio web creado para promover los destinos turísticos de Piura, Perú.

## Características

- Diseño responsive
- Carrusel de imágenes
- Información sobre destinos turísticos
- Formulario de contacto

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
